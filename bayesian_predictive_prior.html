Details

The first plot contains 100 ModSkurt curves. The parameters for each of the curves are independently sampled from a prior distribution. The second plot shows the prior distribution for a given parameter. You can choose which parameter is shown by clicking on its tab just above the plot. The hyper-parameters that specify the prior distribution can be altered by moving the sliders below the second plot.
